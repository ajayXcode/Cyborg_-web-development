# NEXUS | Cybernetic Augmentation

A premium, highly-responsive landing page showcasing a futuristic "cyborg" aesthetic. Built with modern web technologies and a focus on sleek, minimalist design, high-end typography, and subtle ambient animations.

## Features

- **Premium UI/UX**: Minimalist, Apple-esque futuristic design with extreme negative space and refined typography.
- **Ambient Lighting**: CSS-driven drifting light orbs and cinematic film grain overlays.
- **Glassmorphism**: Advanced frosted glass effects for UI panels and data visualizations.
- **Smooth Animations**: Custom CSS bezier-curve transitions, scroll reveals (via Intersection Observer), and typing effects.
- **Fully Responsive**: Adapts seamlessly from ultra-wide desktop monitors down to mobile devices.

## Tech Stack

- **HTML5**: Semantic and clean markup.
- **Vanilla CSS3**: Custom CSS variables, advanced grid/flexbox layouts, keyframe animations, and clip-paths. No external CSS frameworks were used, ensuring maximum control and performance.
- **Vanilla JavaScript**: Lightweight DOM manipulation, typing effects, and `IntersectionObserver` for scroll-triggered animations.
- **Vite**: Ultra-fast frontend tooling for development and bundling.

## Setup & Local Development

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ajayXcode/Cyborg_-web-development.git
   cd Cyborg_-web-development
   ```

2. **Install dependencies:**
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173/` (or whichever port Vite assigns).

4. **Build for production:**
   ```bash
   npm run build
   ```
   The optimized, production-ready files will be output to the `dist/` directory.

## Design Details

- **Typography**: Uses `Space Grotesk` for high-impact, futuristic headings and `Inter` for clean, legible body text.
- **Color Palette**: Deep blacks (`#030303`), sleek surface greys (`#0a0a0a`), and a piercing electric blue accent (`#00E5FF`).
